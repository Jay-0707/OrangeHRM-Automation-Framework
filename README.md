# OrangeHRM Automation Framework

## 📌 Project Overview

This project is a Selenium Automation Testing Framework developed using Java, Selenium WebDriver, TestNG, and Maven following the Page Object Model (POM) design pattern.

The framework automates end-to-end testing of the OrangeHRM application and includes reusable components, data-driven testing, logging, screenshot capture, and cross-browser support.
